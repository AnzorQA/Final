Git

  • git init - инициализировать новый репозиторий.
  • git clone <url> - склонировать удаленный репозиторий.
  • git add . - добавить все изменения в индекс.
  • git commit -m "Текст коммита" - закоммитить изменения.
  • git status - показать статус текущего репозитория.
  • git push - отправить изменения в удаленный репозиторий.
  • git config --global user.name "Ваше имя" - указать имя пользователя.
  • git config --global user.email "your@email.com" - указать email пользователя.
